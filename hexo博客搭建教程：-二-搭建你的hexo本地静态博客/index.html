1. 安装node.js

到node.js官网下载安装文件，并一路Next即可。

2. 安装Git

hexo语法速查：

  • hexo g 　　　#生成
  • hexo d 　　　#部署
  • hexo n 　　　#写文章
  • hexo s 　　　#本地查看
  • hexo d -g 　　#生成–>部署，相当于hexo g与hexo d合并。
  • hexo s -g 　　#生成–>本地查看，相当于hexo g与hexo s合并。
